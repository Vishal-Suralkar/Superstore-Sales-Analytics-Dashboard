# 📊 Superstore Sales Analytics Dashboard (Power BI)

## 🔍 Overview
This project is a 3-page interactive Power BI dashboard built to analyze sales performance, profitability, and loss-making areas using the Superstore dataset.

---

## 📌 Key Features
- KPI Overview (Sales, Profit, Profit Margin)
- Category & Region-wise performance analysis
- Top 10 & Bottom 10 product analysis
- Loss-making segment identification
- Interactive slicers (Region, Category, Segment)

---

## 📈 Dashboard Pages

### 1️⃣ Executive Summary
- Total Sales, Profit, Profit Margin
- High-level performance overview

### 2️⃣ Performance Breakdown
- Sales by Category
- Profit by Sub-category
- Monthly sales trend

### 3️⃣ Insights & Loss Analysis
- Top & Bottom products
- Profit contribution by category
- Loss-making segments

---

## 💡 Key Insights
- Technology category drives highest profit
- Tables and Bookcases are major loss-makers
- West region has high sales but lower profitability
